## 🎯 Overview

Blue Carbon Ledger leverages blockchain, AI, and environmental data to revolutionize how we measure, verify, and trade carbon captured by vital coastal ecosystems — including mangroves, seagrasses, and salt marshes.
By combining decentralized transparency with scientific verification, we’re building a more accountable path toward a sustainable future.

## 👥 Target Users

🌱 NGOs: Manage and monitor conservation projects

🏢 Corporations: Purchase verified, traceable carbon credits

🔬 Scientists: Validate ecological and carbon data

🌍 Government Agencies: Oversee compliance and reporting

👨‍👩‍👧‍👦 Local Communities: Benefit from ecosystem preservation initiatives

## ✨ Key Features

### 🔗 Blockchain-Powered Transparency

- Ethereum-based smart contracts for secure, immutable records

- Carbon credit issuance and transfer via decentralized ledger

### 📡 Real-Time Ecosystem Monitoring

- Integration with satellite and IoT data
- Live analytics dashboard for carbon stock tracking
- AI-driven environmental impact assessments

### 🧠 AI-Powered Verification

- Google Gemini AI for intelligent project evaluation

### 📊 Interactive Dashboards

- Role-based dashboards for NGOs, corporations, and scientists

- Automated project validation and credit transactions
- Transparent, auditable operations

## 🧰 Technology Stack

- Frontend: React 18, TypeScript, Tailwind CSS, shadcn/ui

## 🚀 Installation Guide
1. Clone the Repository
```bash
git clone https://github.com/sanchit-borikar/BLOCK-CHAIN-BASED-BLUE-CARBON-LEDGER.git
cd BLOCK-CHAIN-BASED-BLUE-CARBON-LEDGER
```

> Note: If the Mermaid diagram does not render on your platform, a plain-text fallback is provided below.

### System Architecture (fallback view)

```
User Device
  └─> Blue Carbon Ledger Web App
        ├─ Frontend
        │    ├─ React + TypeScript UI
        │    └─ Three.js Visualizations
        ├─ Authentication
        │    ├─ Supabase Auth
        │    └─ MetaMask / Web3 Wallet
        ├─ Monitoring & AI
        │    ├─ Satellite / IoT Data
        │    └─ AI Verification (Google Gemini) -> Real-time Analytics
        ├─ Storage Layer
        │    ├─ IPFS / Pinata
        │    └─ Off-chain DB (Supabase)
        ├─ Blockchain Layer
        │    └─ Ethereum Smart Contracts (Carbon Credit Ledger)
        └─ Verification & Oracles
             ├─ Verifier Service
             └─ Oracles / Orchestration
```


2. Setup Environment Variables
```bash
cp .env.example .env
```

Edit `.env` with your API keys, wallet, and service credentials.

3. Install Dependencies
```bash
npm install
# or
bun install
```

4. Start the Development Server
```bash
npm run dev
# or
bun dev
```
